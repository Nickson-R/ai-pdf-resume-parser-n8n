# AI PDF Resume Parser using n8n

An AI-powered workflow automation project that extracts text from uploaded PDF resumes, analyzes the content using Groq LLMs, and stores AI-generated feedback automatically in Google Sheets.

---

## Features

- PDF resume upload form
- Automatic PDF text extraction
- AI-powered resume analysis
- ATS-style feedback generation
- Career recommendations
- Google Sheets automation
- End-to-end workflow pipeline

---

## Tech Stack

- n8n
- Groq API
- Google Sheets
- Workflow Automation
- PDF Processing

---

## Workflow Architecture

Upload Resume PDF  
↓  
Extract Resume Text  
↓  
AI Resume Analysis  
↓  
Store Results in Google Sheets
